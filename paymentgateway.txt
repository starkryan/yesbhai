## 🛒 Create Order API

**Endpoint:**  
`https://apnabestupi.in/api/create-order`

**Method:**  
`POST` (application/x-www-form-urlencoded)

### Request Parameters:

| Parameter        | Type    | Required | Description                          |
|------------------|---------|----------|--------------------------------------|
| customer_mobile  | string  | Yes      | Customer's mobile number (optional use 1 if not required) |
| user_token       | string  | Yes      | Your unique API token                |
| amount           | string  | Yes      | Payment amount                       |
| order_id         | string  | Yes      | Unique Order ID (generated by you)   |
| redirect_url     | string  | Yes      | URL where users are redirected after payment |
| remark1          | string  | No       | Extra remarks (optional)             |
| remark2          | string  | No       | Extra remarks (optional)             |

---

### Example PHP Code to Create Order:

```php
<?php
$api_url = 'https://apnabestupi.in/api/create-order';

$order_id = rand(10000, 99999); // Generate random order id

$data = array(
    'customer_mobile' => '1',
    'user_token' => '80e74ccc34adfabe2641836kje6d0fc1c',
    'amount' => '10',
    'order_id' => $order_id,
    'redirect_url' => 'https://yourdomain.com/callback.php',
    'remark1' => 'Online Payment Request',
    'remark2' => 'upb server',
);

// Initialize cURL
$ch = curl_init();
curl_setopt($ch, CURLOPT_URL, $api_url);
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
curl_setopt($ch, CURLOPT_POST, true);
curl_setopt($ch, CURLOPT_POSTFIELDS, http_build_query($data));
curl_setopt($ch, CURLOPT_HTTPHEADER, array('Content-Type: application/x-www-form-urlencoded'));

$response = curl_exec($ch);

if (curl_errno($ch)) {
    echo 'cURL error: ' . curl_error($ch);
} else {
    $result = json_decode($response, true);
    if ($result && isset($result['status']) && $result['status'] === true) {
        $paymentLink = $result['result']['payment_url'];
        echo "Payment Link: " . $paymentLink;
    } else {
        echo 'Error: ' . $result['message'];
    }
}

curl_close($ch);
?>
```

## 🔁 Webhook / Callback Handling

Once payment is completed, your Webhook URL will receive the `order_id`.  
You can then **verify the payment** by using **Check Order Status API**.

Example Callback Handling:

```php
<?php
$order_id = $_POST['order_id']; // Get order_id from webhook

$user_token = "80e74ccc34adfabe2641836ae6d0fc1c";
$api_url = "https://apnabestupi.in/api/check-order-status";

$data = array(
    "user_token" => $user_token,
    "order_id" => $order_id,
);

$options = array(
    'http' => array(
        'header'  => "Content-Type: application/json",
        'method'  => 'POST',
        'content' => json_encode($data),
    ),
);

$context = stream_context_create($options);
$response = file_get_contents($api_url, false, $context);

$result = json_decode($response, true);

if ($result['status'] === 'COMPLETED') {
    // Payment Successful
    // Update your database here
    echo "Payment Successful, UTR: " . $result['result']['utr'];
} else {
    // Payment Failed
    echo "Payment Failed or Pending: " . $result['message'];
}
?>
```

## ⚙️ Notes:

- **Order Timeout:** Orders are valid for **30 minutes**. After that, they **expire automatically**.
- Always **verify payment** using the **Check Order Status API** even if you receive a webhook.
- `order_id` must be **unique** for every payment request.
